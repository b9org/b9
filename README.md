Cloning
-------
To clone the source code, use:
```
git clone https://github.com/youngar/Base9
```

Building
--------
You may not need to disable warnings as errors to build.
```
make
```

To run tests, use:
```
make test
```

To run the benchmark, use:
```
make bench
```
